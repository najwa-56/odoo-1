Fetch Live/Current GPS location from Google Map in Contacts/Customers/Vendors

The "Fetch Live/Current GPS location from Google Map in Contacts/Customers/Vendors" allow users to fetch address from current/live GPS location via Google MAP integration. User can click on "Fetch Address" button at Contact level to open map view and set current/live GPS location and update it on customer address fields.

This module also support multilingual where map is translated in users language.

Compatible with PC & mobile browser only. This will not work in mobile app as odoo mobile app do not support to access live gps location.

Note: Odoo will fetch the address based on data available in google MAP, it could be full address, latitude/longitude.
