[Original] POS Manager Validation using User PIN

Way to have manager validation for specific features in POS

Features:

  • Validate POS Closing
  • Validate Order Deletion
  • Validate Order Line Deletion
  • Validate Order Payment
  • Validate Discount Application
  • Validate Price Change
  • Validate Decreasing Quantity
  • Validate Cash In/Out (Odoo v17 Only)

Important: Manager settings uses User (res.users) table, not Employee (hr.employee) table!

Configuration

Easy setup of manager validation

Manager validation configuration is as easy as ABC. Configuration can be found in Point of Sale > Configuration > Settings.

Specify managers (users with POS / Manager group) in Manager Validation (User) tab for each point of sale. Then select which features in POS you want to have manager validation.

POS Interface

Manager validation seemless integration to POS interface

There will be Manager Validate window that will ask the manager's passcode/password in order to proceed with the action.

Note that the passcode can be configured in Settings > Users & Companies > Users > Point of Sale tab.

Compatibility

This module is compatible with Odoo 17.0 Community and Enterprise Editions
